一、SpringBoot 配置优先级

  • SpringBoot 中支持三种格式的配置文件：

  • 注意：虽然 springboot 支持多种格式配置文件，但是在项目开发时，推荐统一使用一种格式的配置（yml是主流）。

  • SpringBoot 除了支持配置文件属性配置，还支持 Java 系统属性和命令行参数的方式进行属性配置。（命令行参数优先级高于 Java 系统属性）

    • Java 系统属性

    • 命令行参数

  • 项目打包

    1. 执行 maven 打包指令 package

    2. 执行 java 指令，运行 jar 包

  • 注意：SpringBoot 项目进行打包时，需要引入插件 spring-boot-maven-plugin（基于官网骨架创建项目，会自动添加该插件）

  • 总结：

    • 配置优先级：命令行参数 > Java 系统属性 > properties > yml > yaml
