15.欧拉函数|(扩展)欧拉定理|欧拉反演

1.欧拉函数

1）定义

​ 欧拉函数是小于等于 x 的数中与 x 互质的数的数目。

​ 符号 $\varphi (x)$

2）通式

​ $\varphi (x)=x\prod^{n}_{i=1}(1-\frac{1}{p_i})$

​ 其中，$p_i$ 为 x 的质因子，n 为 x 的质因子个数

2.欧拉函数常用性质

1）若 n 为质数，显然 $\varphi (n)=n-1$

2）欧拉函数是积性函数

​ 积性函数：对任意互质的整数 a 和 b，有 $f(ab)=f(a)*f(b)$

​ 若 m，n 互质，则 $\varphi (mn)=\varphi(m)*\varphi(b)$

3）若 $x=2n$ （n 为奇数），$\varphi(x)=\varphi(n)$ 即 $\varphi(2n)=\varphi(x)$

​ n 为奇数时，n 与 2 互质，$\varphi(2)=1$

4）若 p 为质数，则 $\varphi(p^k)=p^k-p^{k-1}$

​ 因为与 $p^k$ 不互质的只有 p 的倍数，而 $p^k$ 中 p 的倍数有 $p^{k-1}$ 个

5）当 $x>2$ 时，$\varphi(x)$ 为偶数

6）小于 n 的数中，与 n 互质的数的总和为 $\frac{\varphi(n)*n}{2}(n>1)$

7）欧拉反演：

​ $n=\sum_{d|n}\varphi(n)$

​ 即 n 的因数（包括 1 和它本身）的欧拉函数之和等于 n。

​ 证明：
$$
设f(x)=\sum_{d|n}\varphi(x)\
\because f(n)*f(m)=\sum_{i|n}\sum_{j|m}\varphi(n)\varphi(m)=\sum_{i|n}\sum_{j|m}\varphi(nm)=\sum_{d|nm}\varphi(d)=f(nm),\
\therefore f(x)是积性函数\
f(p^k)=\varphi(1)+\varphi(p)+\varphi(p^2)+…+\varphi(p_k)=1+(p-1)+(p^2-p)+…(p^k-p^{k-1})=p^k\
又x=p_1^{k_1}p_2^{k_2}p_3^{k_3}…p_m^{k_m}\
\therefore f(n)=f(p_1^{k_1})f(p_2^{k_2})f(p_3^{k_3})…f(p_m^{k_m})=p_1^{k_1}p_2^{k_2}p_3^{k_3}…p_m^{k_m}=n
$$
​ 扩展：

​
$$
\because gcd(i,j)=\sum_{d|gcd(i,j)}\varphi(d)=\sum_{d|i}\sum_{d|j}\varphi(d)\
\therefore \sum_{i=1}^ngcd(i,n)=\sum_{i=1}^n\sum_{d|i}\sum_{d|n}\varphi(d)=\sum_{d|n}\sum_{i=1}^n\sum_{d|i}\varphi(d)=\sum_{d|n}\frac{n}{d}\varphi(d)\
即\sum_{i=1}^ngcd(i,n)=\sum_{d|n}\frac{n}{d}\varphi(d)
$$

3.欧拉定理

若 a，b 互质，则 $a^{\varphi(b)}\equiv1(mod\space b)$

变式：

​ 若 a，b 互质，则 $a^b\equiv a^{b%\varphi(m)}(mod\space m)$

4.扩展欧拉定理

若 $b>\varphi(m)$，即使 a，m 不互质，也有 $a^b\equiv a^{b%\varphi(m)+\varphi(m)}(mod\space m)$

2.哈密顿问题

哈密顿通路：经过图中每个节点且仅经过一次的通路
哈密顿回路：经过图中每个节点且仅经过一次的回路
哈密顿图：存在哈密顿回路的图
竞赛图：每对顶点之间都有一条边相连的有向图，n个顶点的竞赛图称竞赛图
/与欧拉回路对比：欧拉回路是指不重复地走过所有路径的回路；哈密顿回路是指不重复地走过所有点并且最后回到起点的回路
判定：
1、哈密顿通路的判定
若 deg(u) + deg(v) >= n - 1 成立，则图中存在哈密顿通路
2、哈密顿回路的判定：Dirac定理
若 deg(u) + deg(v) >= n，则图中存在哈密顿回路
推论：对于 n >= 3 的无向图，若其中任意一点u的度数 deg(u) >= n / 2，则图中存在哈密顿回路
3、竞赛图

​ 对于n >= 2的竞赛图，一定存在哈密顿通路

兰道定理(Landau)(判定竞赛图)：
将一个竞赛图的每一个点的出度从小到大排序后得到的序列称为竞赛图的比分序列
对于一个长度为n的序列S = (s1 <= s2 <= … <= sn), n >= 1是合法的比分序列，当且仅当： ∀1 <= k <= n, S’ >= k / 2 (s’ = s1 + s2 + … + sk)，且 k = n 时，式子必须取等

5. 01分数规划

求一个分数表达式的最大（最小）值。

​ 例如，求 $\frac{\sum^k_{i=1}a_i}{\sum^k_{j=1}b_j}$ 的最大值

二分法：

​ 限制 $k\ge w$

​ 令 $sum=\sum^k_{i=1}a_i,\space tot=\sum^k_{j=1}b_j,\space k\ge w$，设最优解为 ans，则

​ $\frac{sum}{tot}\le ans$

​ 移项得：

​ $sum-ans*tot\le 0$

​ 即：

​ $\sum^k_{i=1}(a_i-b_i*ans)\le 0$

二分 ans 即可。
