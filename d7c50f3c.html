OpenFeign的使用

一、简介

OpenFeign 客户端是一个 web 声明式 http 远程调用工具，直接可以根据服务名称去注册中心拿到指定的服务 IP 集合，提供了接口和注解方式进行调用，内嵌集成了 Ribbon 本地负载均衡器。

二、SpringCloud 集成 OpenFeign

  1. 引入 starter

  2. 在启动类或者配置类上加 @EnableFeignClients注解

  3. 声明 Feign 接口

三、替换OpenFeign 默认 HTTP 请求框架为 OkHttp

  • 因为 OpenFeign 默认的 Http 请求框架 HttpURLConnection 没有连接池，可能导致在压力大的情况下的性能损失，可以通过替换为 OkHttp 提供连接池解决此问题。

四、使用 OpenFeign 的最佳实践

为了避免在多个微服务之间重复定义 client 接口，我们可以将其抽取出来。这里有两种抽取思路：

  1. 抽取到微服务之外的公共模块
  2. 每个微服务自己抽取一个模块

对比：

  • 方案一抽取更加简单，工程结构也比较清晰，但缺点是整个项目耦合度偏高；
  • 方案二抽取相对麻烦，工程结构相对更复杂，但服务之间耦合度降低。
